Phase 4
The purpose of this phase is to implement business and process requirements based on the business blueprint. The objectives are:
  • Final implementation in the system
  • Unit test, Integration test, System Acceptance test
  • Release of the system for production (live) operation

Entry Criteria

  • Training Material
  • Cutover & Validation Template

Input

  • UAT Test Scripts
  • Validated Test results
  • UAT sign off from customer

Task

  • Migrate to Production
  • PROD Configuration
  • Integrations Go Live
  • Post Go-Live Support
  • Handoff to Client IT Team

Validation

  • Project /Phase Review by the Quality

Exit Criteria

  • Go-Live Sign Off​​
  • Final acceptance from client​
  • Closure Meeting​

Output

  • Go-Live Sign Off​
  • Final acceptance from client
  • Closure Meeting​
